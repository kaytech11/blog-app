Blog App

A full-stack blogging platform built with Node.js, Express, MongoDB, and EJS. Users can register, login, create, edit, and delete blogs with images and videos. The dashboard tracks total blogs and reads per user.

Features

1.User authentication (register/login)

2.Create, edit, delete blogs with images

3.Dashboard showing blog stats and reads

4.Secure routes: only logged-in users can access blogs

5.Responsive UI with Tailwind CSS

Tech Stack

. Backend: Node.js, Express, MongoDB, Mongoose

. Frontend: EJS templates, Tailwind CSS

. Authentication: Express session, bcrypt for password hashing

. File Uploads: Multer (images)
